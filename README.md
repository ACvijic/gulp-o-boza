My setup for front-end themes.

## What it does:

- compiles sass
- minifies css
- minifies js
- minifies images (`.png` & `.jpg`)
- compiles `.njk` files to `.html`
- cleans out `.html` files (for developing)
- browser-sync

## Prereq's:

- Node (+ npm)
- Gulp-cli (installed globally)

## Usage:

1. clone repo
1. run `gulp` to compile sass, css, js & html
